# Auto-Reconnect

If the Controller loses connection to the machine for a reason other than the operator disconnecting it, the software can attempt to reconnect automatically.

## Configuration

The auto-reconnection settings are found in **Settings** <img src="../../.gitbook/assets/image (18).png" alt="" data-size="line">, in the **Controller** section:

<figure><img src="../../.gitbook/assets/image (22).png" alt=""><figcaption></figcaption></figure>
