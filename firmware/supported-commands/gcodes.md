# GCodes

| Gcode                               | function                                                                                                                                                                                                                                                                      | example                 | community firmware only |
| ----------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------- | ----------------------- |
| [G0](https://smoothieware.org/g0)   | Move to the given coordinates. It is used to go to a new area without cutting, it is also called a “rapid” move. Note that the F parameter can be used here and is remembered by subsequent commands                                                                          | G0 X10 Y5 F100          |                         |
| [G1](https://smoothieware.org/g1)   | Move to the given coordinates, see above for the difference with G0. Takes the same F parameter as G0                                                                                                                                                                         | G1 X20 Y2.3 F200        |                         |
| [G2](https://smoothieware.org/g2)   | Clockwise circular motion : go to point with coordinates XYZ while rotating around point with relative coordinates IJ                                                                                                                                                         | G2 X10 J5               |                         |
| [G3](https://smoothieware.org/g3)   | Counter-clockwise motion : see above                                                                                                                                                                                                                                          | G3 Y5 X10 I2            |                         |
| [G4](https://smoothieware.org/g4)   | Dwell P\<seconds>                                                                                                                                                                                                                                                             | G4 P1                   |                         |
| G10                                 | Set workspace coordinates [http://linuxcnc.org/docs/html/gcode/coordinates.html](http://linuxcnc.org/docs/html/gcode/coordinates.html) and [http://linuxcnc.org/docs/html/gcode/g-code.html#gcode:g10-l2](http://linuxcnc.org/docs/html/gcode/g-code.html#gcode:g10-l2)       | G10 L2 P1 X0            |                         |
| [G17](https://smoothieware.org/g17) | Select XYZ plane (command is modal)                                                                                                                                                                                                                                           | G17                     |                         |
| [G18](https://smoothieware.org/g18) | Select XZY plane (command is modal)                                                                                                                                                                                                                                           | G18                     |                         |
| [G19](https://smoothieware.org/g19) | Select YZX plane (command is modal)                                                                                                                                                                                                                                           | G19                     |                         |
| [G20](https://smoothieware.org/g20) | Inch mode : passed coordinates will be considered as Inches, so internally translated to millimeters                                                                                                                                                                          | G20                     |                         |
| [G21](https://smoothieware.org/g21) | Millimeter mode ( default ) : passed coordinates will be considered as millimeters                                                                                                                                                                                            | G21                     |                         |
| G23                                 | select file                                                                                                                                                                                                                                                                   | not used                |                         |
| G24                                 | start print                                                                                                                                                                                                                                                                   | not used                |                         |
| G25                                 | pause print                                                                                                                                                                                                                                                                   | not used                |                         |
| G26                                 | reset print                                                                                                                                                                                                                                                                   | not used                |                         |
| G27                                 | report print progress                                                                                                                                                                                                                                                         |                         |                         |
|                                     |                                                                                                                                                                                                                                                                               |                         |                         |

| G38                                 | [Standard probe commands implemented as documented ](http://linuxcnc.org/docs/2.6/html/gcode/gcode.html#sec:G38-probe)[here](http://linuxcnc.org/docs/2.6/html/gcode/gcode.html#sec:G38-probe)                                                                                | G38.2 Z-10              |                         |
| G38.2                               | probe toward workpiece, stop on contact, signal error if failure                                                                                                                                                                                                              |                         |                         |
| G38.3                               | probe toward workpiece, stop on contact                                                                                                                                                                                                                                       |                         |                         |
| G38.4                               | probe away from workpiece, stop on loss of contact, signal error if failure                                                                                                                                                                                                   |                         |                         |
| G38.5                               | probe away from workpiece, stop on loss of contact,                                                                                                                                                                                                                           |                         |                         |

Hard Disabled in Firmware                                                                                                                                                                                             |                         |                         |

| G54                                 | Use workspace coordinates [http://linuxcnc.org/docs/html/gcode/coordinates.html](http://linuxcnc.org/docs/html/gcode/coordinates.html) and [http://linuxcnc.org/docs/html/gcode/g-code.html#gcode:g54-g59.3](http://linuxcnc.org/docs/html/gcode/g-code.html#gcode:g54-g59.3) | G54                     |                         |
